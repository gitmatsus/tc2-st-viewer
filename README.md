# TwinCAT 2 ST Viewer

Beckhoff社の **TwinCAT 2**で作成されたプロジェクトファイル（`.pro`）を、ブラウザ上で直感的に閲覧・比較するためのポータブルなWebツールです。
PLCの現場やオフィスで、専用のIDE（PLC Control）を立ち上げることなく、素早くコードを確認・比較することを目的として設計されています。

**公開URL**: [https://gitmatsus.github.io/tc2-st-viewer/](https://gitmatsus.github.io/tc2-st-viewer/)

## 🚀 主な機能

- **高機能STビューア**: 
  - 宣言部（VAR...END_VAR）と実装部を2分割で表示するTwinCAT 2ライクなレイアウト。
  - POU（プログラム、関数ブロック、関数）を自動解析し、サイドバーにツリー形式で一覧表示。
  - TwinCAT 2独自のキーワード、データ型、標準関数に対応したシンタックスハイライト。
- **プロジェクト比較 (Diff Mode)**:
  - 2つのプロジェクトファイル（.pro）やエクスポートファイル（.exp/.st）を左右に並べて比較。
  - 行単位の差異（変更・追加・削除）を色分けして表示。
  - 両画面の同期スクロールと、差異箇所へのクイックジャンプ。
- **検索・ナビゲーション**:
  - インクリメンタルサーチ対応。
  - F2キーで選択範囲を検索、F3 / Shift+F3 で次/前のヒット箇所へ移動する直感的な操作。
- **ポータビリティ**:
  - HTMLファイル1つで完結（シングルファイルSPA）。
  - 外部ライブラリへの依存なし。オフラインの現場PCでもブラウザがあれば即座に動作します。

## 対応フォーマット

| 形式 | 拡張子 | 説明 |
|------|--------|------|
| バイナリ | `.pro` | TwinCAT 2 プロジェクトファイル（ツリー構造も解析） |
| テキスト | `.st` | ST（Structured Text）ソースファイル |
| テキスト | `.exp` | エクスポートファイル |

## ビューアモード

### プロジェクトツリー

- **階層表示**: POUs セクション（フォルダ + PRG/FB/FUN）と Resources セクションをツリー形式で表示
- **折り畳み**: セクションヘッダーのクリックで展開/折り畳み
- **検索ヒットバッジ**: 各ファイルの横に検索マッチ件数を表示

### コード表示

- **宣言部/実装部 分割表示**: TwinCAT 2 エディタと同じ上下レイアウト
- **リサイザー**: 宣言部と実装部の境界をドラッグしてサイズ調整可能
- **行番号**: TwinCAT 準拠の行番号表示（スクロール同期）
- **シンタックスハイライト**: 予約語・データ型・関数・文字列・数値・コメントを色分け
- **ダブルクリック**: 単語を選択し、同じ単語を全箇所ハイライト

### 検索

- **プロジェクト横断検索**: 全 POU を対象にリアルタイム検索
- **検索オプション**: 大文字/小文字の区別（Aa）、単語単位検索（W）
- **ナビゲーション**: ▲▼ ボタンまたは F3 / Shift+F3 でマッチ間を移動
- **タブ・ツリー連動**: 各ファイルのヒット件数をタブバーとサイドバーに表示

## 比較モード

2つのファイルを並べて差分を確認できます。

### ファイル読み込み

- 「ファイル1」「ファイル2」ボタンから選択、またはドラッグ＆ドロップ
- ドロップゾーンが左右に分かれており、直感的にファイルを配置可能

### 比較結果ツリー

- **階層表示**: ビューアと同じ POUs / Resources のフォルダ構造
- **ステータス表示**: 各 POU の状態をバッジで表示
  - `~ 変更 (N)` — 変更あり（差分行数を括弧内に表示）
  - `+ 追加` — 片方のファイルにのみ存在（追加）
  - `- 削除` — 片方のファイルにのみ存在（削除）
  - `= 同一` — 差分なし
- **折り畳み**: セクションごとに展開/折り畳み可能

### 差分表示

- **左右並列表示**: 宣言部・実装部それぞれで左右に並べて比較
- **行レベル差分**: LCS アルゴリズムによる正確な差分検出
- **色分け**: 変更（黄）、削除（赤）、追加（緑）、プレースホルダー（灰）
- **同期スクロール**: 左右ペインのスクロールが連動
- **リサイザー**: 宣言部と実装部の表示比率をドラッグで調整可能

### 差分マップ

- 宣言部・実装部それぞれの右端にミニマップを表示
- 変更箇所の位置を色付きバーで可視化
- ビューポート表示（現在表示中の範囲を青枠で表示）
- クリック＆ドラッグで任意の位置にスクロール

### 差分ナビゲーション

- ヘッダー右端の **差分 ▲▼** ボタンで変更箇所間をジャンプ
- 現在位置を「1/3箇所」形式で表示
- コードをクリックした位置を起点に、最寄りの変更箇所へ移動

### 比較モード検索

- ビューアと同じ検索機能が比較モードでも利用可能
- 左右ペインをクリックして検索対象の切り替え
- クリック位置を起点とした検索ナビゲーション

## キーボードショートカット

| キー | 動作 |
|------|------|
| **F2** | 選択中のテキストを検索ボックスに取り込み |
| **F3** | 次の検索結果へジャンプ |
| **Shift+F3** | 前の検索結果へジャンプ |
| **Enter** | 次の検索結果（検索ボックスにフォーカス時） |
| **Escape** | 検索をクリア |

## 使い方

### ビューアモード

1. `tc2_viewer.html` をブラウザで開きます
2. 「ファイルを開く」ボタン、またはドラッグ＆ドロップでファイルを読み込みます
3. サイドバーのツリーから POU を選択して閲覧します

### 比較モード

1. ヘッダーの「比較」ボタンをクリックしてモードを切り替えます
2. ファイル1・ファイル2 をそれぞれ読み込みます（ボタンまたはドラッグ＆ドロップ）
3. 左側の比較結果ツリーから POU を選択して差分を確認します
4. 差分 ▲▼ ボタンで変更箇所間を移動できます

## 技術情報

- **依存関係なし**: 外部ライブラリ不使用、HTML/CSS/JavaScript のみ
- **シングルファイル**: `tc2_viewer.html` 1ファイルで完結
- **バイナリ解析**: `.pro` ファイルのバイナリツリー構造を解析し、フォルダ階層を復元
- **差分アルゴリズム**: LCS（最長共通部分列）ベースの行レベル差分
- **文字コード**: CP932 (Shift-JIS) → UTF-8 自動変換
