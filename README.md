# TwinCAT 2 ST Viewer

TwinCAT 2 のプロジェクトファイルを、専用ソフトなしでブラウザ上で閲覧・解析するためのシングルファイル HTML アプリケーションです。

## 主な機能

### 📂 ファイル解析とデコード

- **マルチフォーマット対応**: `.pro` (バイナリ), `.st`, `.exp` の読み取りに対応。
- **日本語エンコード修復**: Shift-JIS (CP932) 自動デコードにより、コメントの文字化けを防止。
- **バイナリクリーンアップ**: 純粋なソースコード（ST 言語）のみを抽出。

### 🔍 高度な検索・閲覧

- **プロジェクト横断検索**: プロジェクト全体から文字列を検索し、ヒット件数を可視化。
- **シンタックスハイライト**: ST 言語の予約語やデータ型を色分け表示。
- **POU 構造の自動抽出**: PRG / FB / FUN をツリー形式で表示。

### ⚡ ショートカット

- **F2**: 選択した単語を検索ボックスへ取り込み。
- **F3 / Shift+F3**: 次の検索結果 / 前の検索結果へジャンプ。

## 使い方

1. `tc2_viewer.html` をブラウザで開きます。
2. 画面上の「ファイルを開く」ボタン、または画面中央へファイルをドラッグ＆ドロップしてください。
