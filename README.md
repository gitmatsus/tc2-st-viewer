# TwinCAT 2 ST Viewer

Beckhoff社の **TwinCAT 2** で作成されたプロジェクトファイル（.pro）や構造化テキスト（ST）を、ブラウザ上で直感的に閲覧・比較するためのポータブルなWebツールです。

PLCの現場やオフィスで、専用のIDE（PLC Control）を立ち上げることなく、素早くコードを確認・比較することを目的として設計されています。

**🔗 公開URL:** [https://gitmatsus.github.io/tc2-st-viewer/](https://gitmatsus.github.io/tc2-st-viewer/)

## 🚀 主な機能

### 高機能STビューア
- **2分割レイアウト**: 宣言部（VAR...END_VAR）と実装部を上下に分けて表示する、TwinCAT 2ライクなインターフェース。
- **POU自動解析**: プログラム、関数ブロック、関数を解析し、サイドバーにフォルダ階層を含むツリー形式で一覧表示。
- **シンタックスハイライト**: TwinCAT 2独自のキーワード、データ型、標準関数に対応。

### プロジェクト比較 (Diff Mode)
- **並列比較**: 2つのプロジェクト（.pro）やエクスポートファイル（.exp/.st）を左右に並べて比較。
- **行レベル差分**: LCSアルゴリズムにより、変更・追加・削除を正確に色分け表示。
- **高度な同期**: 左右ペインの同期スクロールに加え、差分箇所へのクイックジャンプ機能を搭載。

### 検索・ナビゲーション
- **インクリメンタルサーチ**: 入力に合わせて即座にヒット箇所を表示。
- **エディタ風操作**: F2で選択ワードを検索、F3で次へ移動など、直感的なショートカットに対応。

### ポータビリティ
- **シングルファイルSPA**: `tc2_viewer.html` 1ファイルで完結。
- **完全オフライン動作**: 外部ライブラリへの依存がなく、ネット環境のない現場のPCでもブラウザがあれば動作します。

## 🛠 使い方

### ビューアモード
1. `tc2_viewer.html` をブラウザで開きます。
2. 「ファイルを開く」ボタン、またはドラッグ＆ドロップでファイルを読み込みます。
3. サイドバーのツリーから目的の POU を選択して閲覧します。

### 比較モード
1. ヘッダーの「比較」ボタンをクリックしてモードを切り替えます。
2. ファイル1・ファイル2 をそれぞれ読み込みます。
3. 左側の比較結果ツリーから、ステータスバッジ（`~`, `+`, `-`）を参考に差分を確認します。
4. ヘッダーの **差分 ▲▼** ボタンで変更箇所間を素早く移動できます。

---

## ⌨️ ショートカットキー

| キー | 動作 |
| :--- | :--- |
| **F2** | 選択中のテキストを検索ボックスに取り込み |
| **F3** | 次の検索結果へジャンプ |
| **Shift + F3** | 前の検索結果へジャンプ |
| **Enter** | 次の検索結果（検索ボックスにフォーカス時） |
| **Escape** | 検索をクリア（検索ボックスにフォーカス時） |

## 🔍 詳細仕様

### 📂 対応ファイル形式
- `.pro` (TwinCAT 2 プロジェクトファイル)
- `.exp` (書き出し済みPOU)
- `.st` (構造化テキストファイル)

### 🌳 比較結果ツリー
- **階層表示**: ビューアと同じ `POUs` / `Resources` のフォルダ構造を維持。
- **ステータスバッジ**:
  - `~ 変更 (N)` — 変更あり（カッコ内は差分行数）
  - `+ 追加` — 片方のファイルにのみ存在（追加）
  - `- 削除` — 片方のファイルにのみ存在（削除）
  - `= 同一` — 差分なし

### 🗺️ 差分表示とマップ
- **同期スクロール**: 左右のペインが連動して動き、コードの対応関係を保持。
- **リサイザー**: 宣言部と実装部の表示比率をドラッグで自由に調整可能。
- **差分ミニマップ**: 右端のマップでファイル全体の変更箇所の分布を可視化。クリックやドラッグで任意の位置へジャンプ。

## ⚙️ 技術情報

- **ノーディペンデンシー**: 外部ライブラリ不使用。Vanilla JS / CSS3 のみで構築。
- **バイナリ解析**: `.pro` ファイルのバイナリツリー構造を解析し、独自のフォルダ階層を復元します。
- **差分アルゴリズム**: LCS（最長共通部分列）ベースの行レベル差分検出。
- **文字コード対応**: 日本語環境（CP932/Shift-JIS）から UTF-8 への自動変換機能を内蔵。
